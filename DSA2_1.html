DSA II: Page 1 - Quick Sort

The Efficient "In-Place" Sorter

Quick Sort is one of the most popular and efficient sorting algorithms, frequently used in language standard libraries (like sort() in many languages). Like Merge Sort, it is a divide-and-conquer algorithm. However, unlike Merge Sort, Quick Sort is typically in-place—meaning it doesn't require massive extra memory to hold subarrays—which makes it highly efficient in memory-constrained environments.

1. The Algorithm: Partitioning

Quick Sort relies on a "partitioning" operation to sort data:

  1. Pick a Pivot: Choose one element from the array to be the "pivot."
  2. Partitioning: Reorder the array so that all elements less than the pivot are on the left, and all elements greater than the pivot are on the right.
  3. Conquer: Recursively apply the same logic to the left and right subarrays.

def quick_sort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    return quick_sort(left) + middle + quick_sort(right)

2. Complexity Analysis

  • Average Case: O(n log n). If the pivot consistently divides the array into relatively equal halves, it is extremely fast.
  • Worst Case: O(n²). This occurs if you pick a poor pivot (like the smallest or largest element) every time, leaving one side of the partition empty.

Real-World Relationship: Organizing a Bookshelf

Think of Quick Sort like organizing a disorganized bookshelf. You pick a book from the shelf at random—let's call it the Pivot. You then go through all the other books and put every book that starts with a letter before the pivot's title on the left side of the table, and every book after on the right. Now, the pivot book is in its final, correct spot! You then repeat this exact same process for the pile of books on the left and the pile of books on the right. Because you are organizing them *around* a known reference point, the shelves fill up very quickly.

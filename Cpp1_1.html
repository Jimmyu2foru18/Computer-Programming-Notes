C++ I: Page 1 - Introduction to C++ and the Development Environment

What is C++?

C++ is a high-performance, compiled, object-oriented language that provides low-level memory manipulation features while maintaining high-level abstractions. It is the powerhouse behind operating systems, game engines, high-frequency trading platforms, and large-scale enterprise software.

The C++ Development Pipeline

Unlike Python, C++ must be compiled into machine code before it can be executed. This process involves:

  1. Preprocessing: Handling macros and including header files.
  2. Compilation: Translating source code into object files.
  3. Linking: Combining object files into an executable.

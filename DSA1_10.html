DSA I: Page 10 - Merge Sort

The Efficient Divide-and-Conquer

Merge Sort is a highly efficient, stable, and predictable sorting algorithm. Unlike Bubble or Insertion Sort, which perform poorly on large datasets (O(n²)), Merge Sort consistently performs at O(n log n) time complexity. It achieves this by using a divide-and-conquer strategy: it recursively splits the array into halves until it has a bunch of single-element arrays (which are inherently sorted), and then it merges those small arrays back together in the correct order.

1. The Algorithm

The magic happens in the "merge" phase:

  1. Divide: Find the midpoint and split the array into two halves.
  2. Conquer: Recursively call Merge Sort on both halves until the base case (single-element arrays) is reached.
  3. Merge: Take two sorted halves and merge them back into a single sorted array by comparing the smallest elements of each half repeatedly.

def merge_sort(arr):
    if len(arr) > 1:
        mid = len(arr) // 2
        left = merge_sort(arr[:mid])
        right = merge_sort(arr[mid:])
        return merge(left, right)
    return arr

2. Why It Matters

Merge sort is stable (it preserves the relative order of equal elements) and efficient. Its only major trade-off is space complexity: it requires O(n) extra space to hold the subarrays during the merging process, making it less memory-efficient than algorithms that sort "in-place."

Real-World Relationship: The Book Organizer

Think of Merge Sort like organizing a massive library. Instead of one person trying to sort a thousand books alone, you divide the books into two stacks. You give each stack to a friend and tell them to split their stack in half again. Eventually, you have many tiny stacks of one book each. Now, you and your friends work together to merge these tiny stacks back into larger, sorted stacks, comparing the first book of each tiny stack and placing the smaller one first. Because everyone is working on smaller, manageable parts simultaneously, the massive library is organized in a fraction of the time.

Web Technologies I: Page 4 - The CSS Box Model & Layout

Everything is a Box

In CSS, every single element on the page is treated as a rectangular box. Understanding how these boxes are sized, spaced, and positioned is the single most important skill for mastering web layouts. If you don't understand the Box Model, you will constantly be frustrated by elements not aligning correctly or taking up the wrong amount of space.


.box {
  width: 300px;
  border: 1px solid black;
  padding: 10px;
}

1. The Box Model Anatomy

Each box consists of four layers, working from the inside out:

  • Content: The text, image, or media inside the element.
  • Padding: Transparent space inside the border, between the content and the border.
  • Border: A line that surrounds the content and padding.
  • Margin: Transparent space outside the border, separating this box from other boxes.

/* The Box Model */
.box {
  width: 200px;
  padding: 20px;
  border: 5px solid black;
  margin: 10px;
  box-sizing: border-box; /* Crucial: includes padding/border in width */
}

2. Layout Systems: Flexbox & Grid

Before, we used floats (a nightmare). Now, we have two powerful layout systems:


.flex-container {
  display: flex;
  justify-content: center;
}
  • Flexbox: Best for one-dimensional layouts (a row of buttons, a navigation bar, centering items).
  • Grid: Best for two-dimensional layouts (the overall structure of the entire page: header, sidebar, main content, footer).

Real-World Relationship: The Picture Frame

Think of an HTML element like a Photograph. The actual photo is the Content. The matting around the photo is the Padding (space inside the frame). The frame itself is the Border. The space between this framed photo and the next photo hanging on the wall is the Margin. Flexbox is like organizing these framed photos in a straight line on a shelf. Grid is like deciding how to hang them all on a massive wall—placing them precisely in rows and columns.


/* The Picture Frame Analogy */
.photo-frame {
  padding: 20px;   /* Matting */
  border: 10px solid #5d4037; /* Frame */
  margin: 15px;    /* Space between frames */
  background: white;
}

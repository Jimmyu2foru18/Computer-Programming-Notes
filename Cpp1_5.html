C++ I: Page 5 - Loops: For & While

Automating Tasks with Repetition

Loops allow you to repeat actions. C++ supports for, while, do-while, and the modern range-based for loop (C++11 and later).

The Code implementation


/*
 * File: loops.cpp
 * Demonstration of various C++ looping mechanisms
 */
#include <iostream>
#include <vector>

using namespace std;

int main() {
    // 1. Traditional For Loop
    cout << "For loop:" << endl;
    for (int i = 0; i < 3; i++) {
        cout << "  Iteration " << i << endl;
    }

    // 2. While Loop
    cout << "While loop:" << endl;
    int j = 0;
    while (j < 3) {
        cout << "  Iteration " << j << endl;
        j++;
    }

    // 3. Do-While Loop (Always runs at least once)
    cout << "Do-While loop:" << endl;
    int k = 0;
    do {
        cout << "  Iteration " << k << endl;
        k++;
    } while (k < 3);

    // 4. Range-based For Loop (C++11)
    vector<int> nums = {10, 20, 30};
    cout << "Range-based loop:" << endl;
    for (int n : nums) {
        cout << "  Value: " << n << endl;
    }

    return 0;
}

/*
 * Loop Nuances:
 * 1. 'break' exits the loop immediately.
 * 2. 'continue' skips to the next iteration.
 * 3. Range-based loops are safer and more concise for containers.
 * 4. Infinite loops (e.g., while(true)) require a 'break' condition.
 * 5. Do-while guarantees at least one execution.
 */

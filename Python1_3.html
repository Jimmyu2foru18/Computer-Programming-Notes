Numeric Types: Precision and Science

In Python, numbers are not just simple digits. They are sophisticated objects that can represent everything from the number of items in a cart to the trajectory of a spacecraft. Python provides three main numeric types: Integers, Floating-Point Numbers, and Complex Numbers.

1. Integers (int)

Integers are whole numbers. One of Python's most powerful features is that integers have arbitrary precision—they can be as large as your computer's RAM allows. For readability, Python allows you to use underscores as visual thousands-separators.


count = 42
negative = -17
big_number = 1_000_000  # Python ignores the underscores

2. Floating-Point Numbers (float)

Floats represent decimal numbers. They are stored using the IEEE 754 standard, which means they are extremely fast but can occasionally have tiny precision errors. Python also supports Scientific Notation for very large or very small floats.


pi = 3.14159
scientific = 2.5e-4  # This is 0.00025

3. Complex Numbers (complex)

Commonly used in electrical engineering and physics, complex numbers consist of a real part and an imaginary part (denoted with a j in Python).


complex_num = 3 + 4j
print(complex_num.real)  # 3.0
print(complex_num.imag)  # 4.0

Boolean Type: The Logic of Programming

The Boolean (bool) data type has only two values: True and False. These are the "On" and "Off" switches of your code. In Python, Boolean values are actually a subclass of integers, where True is 1 and False is 0.

Boolean Operations

You can combine Boolean values using logical operators to create complex decision-making rules.

  • and: Returns True only if both sides are true.
  • or: Returns True if at least one side is true.
  • not: Inverts the value (True becomes False).

is_active = True
is_complete = False

print(is_active and is_complete)  # False
print(is_active or is_complete)   # True
print(not is_active)              # False

Real-World Relationship: The Digital Scale and the Security System

Think of Integers like the Items on a Digital Scale—you either have 1 item or 2, there is no "in-between." Think of Floats like the Weight Reading on that scale, which can be 1.5 lbs or 1.50000001 lbs depending on precision. Complex Numbers are like the Radio Waves passing through the air around the scale—they have two dimensions (amplitude and phase) that require complex math to understand. Finally, think of Booleans as the Security System for the room: is_door_locked is either True or False. If is_door_locked and is_alarm_on, then the system is secure. One wrong switch (Boolean value) can change the state of the entire building.

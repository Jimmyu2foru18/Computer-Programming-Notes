Networks I: Page 8 - IP Addressing: The Logic of Location

The Digital Address

Every device on an IP-based network requires a unique identifier—an IP Address. Think of an IP address as the physical mailing address for a device. Just as a letter won't arrive without a street address, data won't arrive without an IP address. IPv4 (Internet Protocol version 4) is the most widely deployed addressing system today. It uses a 32-bit address, typically written in "dotted-decimal" notation (e.g., 192.168.1.1), where each of the four sections is a number from 0 to 255.

1. IP Address Classes

To organize the internet, addresses were historically divided into classes based on the size of the network:

  • Class A: Used for massive networks (government, ISPs). 1-126.
  • Class B: Medium-sized networks (universities, large companies). 128-191.
  • Class C: Small networks (home and small business). 192-223.
  • Class D/E: Reserved for special purposes like Multicasting and experimental research.
Expert Note: In modern networking, "Classful" addressing is mostly obsolete, replaced by CIDR (Classless Inter-Domain Routing), which allows for much more flexible allocation of addresses.

2. Private vs. Public IP Addresses

Not every device needs a unique address on the public internet. Private IP Addresses are reserved for use within your own home or corporate network. They are not routable on the public internet. This conserves the limited supply of public IPv4 addresses and enhances security.


Class A Private: 10.0.0.0/8
Class B Private: 172.16.0.0/12
Class C Private: 192.168.0.0/16

3. CIDR: The Modern Standard

CIDR notation (e.g., 192.168.1.0/24) replaces the rigid address classes. The number after the slash (the Prefix) tells you how many bits of the address belong to the "Network" portion. A `/24` means the first 24 bits (the first three numbers) are the network, and the remaining 8 bits are for the hosts.

Real-World Relationship: The Apartment Building and the Unit Number

Think of an IP Address like the Street Address of an Apartment Building. The "Network" portion of the IP is the Street Name (e.g., "192.168.1"). All the units on that street share the same beginning. The "Host" portion is the Unit Number (e.g., ".101"). Think of Private IPs like your Internal Room Numbers. If you call "Room 202" in your own building, you know who you mean. But if you call "Room 202" at a totally different apartment complex across town, you get the wrong person. A Router (with NAT - Network Address Translation) is like the building concierge—they take your internal room number and represent your house to the outside world using one single, official public street address.

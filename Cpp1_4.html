C++ I: Page 4 - Control Flow: Conditionals

Making Decisions

C++ uses if, else if, else, and switch to control the flow of execution based on conditions. Understanding conditional branching is the foundation of creating dynamic and reactive programs.

The Code implementation


/*
 * File: control_flow.cpp
 * Demonstration of if-else and switch control structures
 */
#include <iostream>

using namespace std;

int main() {
    int score = 85;
    
    // 1. If-Else-If Structure
    if (score >= 90) {
        cout << "Grade: A" << endl;
    } else if (score >= 80) {
        cout << "Grade: B" << endl;
    } else if (score >= 70) {
        cout << "Grade: C" << endl;
    } else {
        cout << "Grade: F" << endl;
    }

    // 2. Switch Statement
    int day = 3; // Wednesday
    switch (day) {
        case 1: cout << "Monday" << endl; break;
        case 2: cout << "Tuesday" << endl; break;
        case 3: cout << "Wednesday" << endl; break;
        case 4: cout << "Thursday" << endl; break;
        case 5: cout << "Friday" << endl; break;
        default: cout << "Weekend" << endl;
    }

    // 3. Logical combinations
    bool isStudent = true;
    bool hasID = false;
    
    if (isStudent && hasID) {
        cout << "Access Granted." << endl;
    } else if (isStudent || hasID) {
        cout << "Access Conditional." << endl;
    } else {
        cout << "Access Denied." << endl;
    }

    return 0;
}

/*
 * Control Flow Nuances:
 * 1. Switch cases require a 'break' to prevent fall-through.
 * 2. 'default' in switch acts as the final 'else'.
 * 3. Logical && and || use "short-circuit" evaluation:
 *    - In (A && B), if A is false, B is NOT evaluated.
 *    - In (A || B), if A is true, B is NOT evaluated.
 * 4. This short-circuiting can be used for performance and safety 
 *    (e.g., checking for null pointers before dereferencing).
 */
